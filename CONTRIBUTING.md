# Contribuindo para o Cogitura

Obrigado por considerar contribuir com o Cogitura! Este é um projeto de pesquisa aberto e toda contribuição é bem-vinda.

## Como Contribuir

### Reportar Bugs

Se você encontrar um bug, por favor abra uma issue com:
- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs atual
- Ambiente (OS, Python version, etc)
- Logs relevantes

### Sugerir Melhorias

Para sugerir melhorias:
- Abra uma issue descrevendo a funcionalidade
- Explique por que seria útil
- Forneça exemplos de uso se possível

### Pull Requests

1. Fork o projeto
2. Crie uma branch (`git checkout -b feature/MinhaFeature`)
3. Faça suas alterações
4. Adicione testes se aplicável
5. Execute os testes (`pytest`)
6. Commit suas mudanças (`git commit -m 'Adiciona MinhaFeature'`)
7. Push para a branch (`git push origin feature/MinhaFeature`)
8. Abra um Pull Request

### Diretrizes de Código

- Siga PEP 8
- Use type hints
- Documente funções e classes
- Adicione testes para novas funcionalidades
- Mantenha o código limpo e legível

### Executar Testes

```bash
pytest tests/
pytest tests/ --cov=cogitura  # Com cobertura
```

### Código de Conduta

- Seja respeitoso e inclusivo
- Aceite críticas construtivas
- Foque no que é melhor para a comunidade
- Mostre empatia com outros membros

## Áreas que Precisam de Ajuda
